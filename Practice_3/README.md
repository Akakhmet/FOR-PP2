## âœ… Topics Summary

### 1) Functions
- Creating basic functions (`def`)
- Using arguments (parameters)
- Returning values (`return`)
- Flexible arguments: `*args`, `**kwargs`

### 2) Lambda
- Short anonymous functions: `lambda`
- Using with:
  - `map()` (transform each element)
  - `filter()` (keep elements by condition)
  - `sorted()` (sort using custom key)

### 3) Classes
- Defining classes and creating objects
- Using `__init__` constructor
- Writing class methods
- Understanding class variables vs instance variables

### 4) Inheritance
- Basic inheritance (child gets parent methods)
- `super()` usage
- Method overriding (child replaces parent method)
- Multiple inheritance (child inherits from two parents)
